Explanations of the Dataset

This dataset shows the gender parity index for gross enrolment ratio(Tertiary education).
It contains data from Albania,Algeria,Argentina,Armenia,Australia,Belgium,Chile,China,Cuba,Indonesia,Ireland.
The time period is from 2012.
The dataset is collected by UNESCO Institute for Statistics(UIS) and given by UNdata.

Description

Chart 1 is a bubble cloud which is made by Inkscape. Each bubble represents a country.
The radius of each bubble are mapped with the observation value of gender parity index for gross enrolment by the formula below:

Radius = (ObservationValue - 0.8)*100 (mm)

Example:
(Radius of China = (1.11584-0.8)*100 = 31.584)
(1.11584 is China's observation value)
